How to Edit

Select a tab to edit that app's formula sections.

Each tab has sections (e.g. "Filtering & Lookup") that contain cards. Click a section header to expand it.

Use + Add Card to add a new formula entry. Use the to edit an existing one.

The Shortcuts tab lets you add/edit keyboard shortcuts per application group.

Click Save to GitHub when done — changes go live on your site immediately.
