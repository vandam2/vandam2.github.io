Center x and y properties

X = (Parent.Width - Self.Width) / 2
Y = (Parent.Height - Self.Height) / 2

If you only want it centered horizontally:

X = (Parent.Width - Self.Width) / 2

If you only want it centered vertically:

Y = (Parent.Height - Self.Height) / 2

Self.Width and Self.Height are the control’s own size, so this places the control in the exact middle of its parent container.
